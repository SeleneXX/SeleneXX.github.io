给定单向链表的头指针和一个要删除的节点的值，定义一个函数删除该节点。
没什么好说的，简单的单向不含表头结点的链表，只需在开头加一个判断要删的是不是头节点即可

````python
class Solution:
    def deleteNode(self, head: ListNode, val: int) -> ListNode:
        if head.val == val:
            head.val = head.next.val
            head.next = head.next.next
        else:
            now = head
            while True:
                if now.next == None:
                    break    
                if now.next.val == val:
                    now.next = now.next.next
                    break            
                now = now.next
        return head
````
